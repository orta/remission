### Image assets

These image assets are placed into `Pod/assets` via `react-native bundle`.

You should not add them here manually, add images inside the relevant folders and then the bundling will happen
as someone makes an Emission deploy. For more context see

* [Artsy Slack](https://artsy.slack.com/archives/C02BAQ5K7/p1500315723167989)
* [artsy/emission#673](https://github.com/artsy/emission/pull/673)
